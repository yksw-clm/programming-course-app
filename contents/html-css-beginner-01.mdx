## HTMLとは？

**HTML**（HyperText Markup Language）は、Webページの構造を作るための言語です。
文章に「ここは見出し」「ここは段落」といった意味を付けていきます。

<Callout type="info">
HTMLは「マークアップ言語」と呼ばれ、プログラミング言語とは少し違います。難しく考えず、文章に目印を付けるイメージで始めましょう！
</Callout>

## 最初のHTML

まずは、最もシンプルなHTMLを書いてみましょう。

```html showLineNumbers
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>私の自己紹介</title>
</head>
<body>
    <h1>こんにちは！</h1>
    <p>私の名前は山田太郎です。</p>
</body>
</html>
```

### コードの解説

- `<!DOCTYPE html>`: これからHTMLを書きますよ、という宣言
- `<html lang="ja">`: HTML文書の始まり。`lang="ja"`で日本語を使うことを示します
- `<head>`: ページの設定情報を書く場所
- `<meta charset="UTF-8">`: 文字コードの設定（日本語を正しく表示するために必要）
- `<title>`: ブラウザのタブに表示されるタイトル
- `<body>`: 実際にページに表示される内容を書く場所
- `<h1>`: 見出し（Heading 1）
- `<p>`: 段落（Paragraph）

## 実際に書いてみよう

### ステップ1: ファイルを作成する

1. テキストエディタを開く（メモ帳でもOKですが、Visual Studio Codeなどがおすすめ）
2. 新しいファイルを作成
3. `index.html`という名前で保存

<Callout type="warning">
ファイル名は必ず **半角英数字** で、拡張子は `.html` にしましょう！
</Callout>

### ステップ2: コードを書く

上記のコードをコピーして、`index.html`に貼り付けてみましょう。
名前の部分は自分の名前に変えてください。

### ステップ3: ブラウザで開く

作成した`index.html`ファイルをダブルクリックすると、ブラウザで開けます。
「こんにちは！」という見出しと、名前が表示されるはずです。
