## はじめに

ようこそ！今日から一緒に、あなただけのWebサイトを作っていきましょう 🎉

プログラミングと聞くと難しそうに聞こえるかもしれませんが、実は **「お願いごとの手紙」** を書くのと似ています。

準備はいいですか？ さっそく、コンピュータにお願いを出してみましょう！

---

## 🚀 まずはコピペで動かしてみよう

右側のエディター（文字を入力するところ）に、下の英語のような文字を**そのままコピーして貼り付けて**みてください。

細かいことは気にしなくてOKです！

```html
<h1>はじめまして！</h1>
<p>私はプログラミング初心者です。</p>

<h2>好きな食べ物</h2>
<p>サーモンが好きです。</p>
```

貼り付けたら、右上のプレビュー画面を見てみましょう。

・・・どうなりましたか？

<Callout type="info">
文字が表示されましたか？✨
「はじめまして！」が**大きく**、「好きな食べ物」が**少し小さく**表示されていたら大成功です！
</Callout>

---

## 🤔 なんで文字の大きさが違うの？

不思議ですよね。ただ文字を書いただけなのに、大きさが勝手に変わりました。

実は、**文字の前後にある記号**が、コンピュータへの「指令」になっているんです。

### 1. `<h1>` は「タイトルにして！」という指令

```html
<h1>はじめまして！</h1>
```

この `<h1>` と `</h1>` は、**「ここからここまでが、一番大きな見出し（タイトル）だよ！」** と教えてあげるマークです。

サンドイッチみたいに、中身をパン（タグ）で挟むイメージです 🥪

### 2. `<h2>` は「小見出しにして！」という指令

```html
<h2>好きな食べ物</h2>
```

`<h2>` は、タイトルの次に大事な「小見出し」を作ります。本で言うところの「章のタイトル」みたいなものです。

### 3. `<p>` は「普通の文章だよ」という指令

```html
<p>私は プログラミング初心者 です。</p>
```

`<p>` は Paragraph（パラグラフ＝段落）の略です。「ここは普通の文章だよ～」と教えてあげています。

---

## ✍️ あなたの自己紹介に変えてみよう

仕組みがわかったところで、さっきの文章を**あなたのこと**に書き換えてみましょう！

**ミッション:**
1. 「プログラミング初心者」を「あなたの名前」に変える
2. 「好きな食べ物」を実際に好きなものに変える

```html
<h1>こんにちは！</h1>
<p>私の名前は [ここに名前を入れる] です。</p>

<h2>好きなこと</h2>
<p>私は [好きなこと] が大好きです！</p>
```

<Callout type="warning">
**注意ポイント ⚠️**
前後の < や > の記号をうっかり消さないように気をつけてください。
この記号がなくなると、コンピュータが「どこまでがタイトル？」と迷ってしまいます。
</Callout>

## 🎉 今日のおさらい

お疲れ様でした！今日は3つの「魔法の記号（タグ）」を使いました。

1. \<h1\> … ページで一番目立つ**大見出し**を作る
2. \<h2\> … その次の**小見出し**を作る
3. \<p\> … **普通の文章**を書く

たったこれだけで、もうあなたはWebページを作れるようになりました。すごいです！👏
