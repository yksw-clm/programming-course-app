## 📦 2種類の「余白」を使い分けよう

余白には、実は2つの種類があります。
**「内側の余白」** と **「外側の余白」** です。

ちょっとイメージしにくいですよね？実際にコードを触りながら試してみましょう！

### 1. 内側の余白：Padding（パディング）

まずは、背景色をつけたタイトルの「内側」を広げてみましょう。
**padding** を使います。

```html
<h2 style="background-color: pink;">狭いタイトル</h2>

<h2 style="background-color: skyblue; padding: 20px;">
  ふんわりしたタイトル
</h2>
```

<Callout type="info">
水色のタイトルの方が、文字の周りにスペースがあって読みやすくないですか？
**20px**（ピクセル）という数字を大きくすると、もっと広くなりますよ！
</Callout>

**イメージ:** ダンボールの中に緩衝材（プチプチ）を詰めて、中身を守る感じです 🎁

### 2. 外側の余白：Margin（マージン）

次は、要素と要素の「間」を空けてみましょう。
**margin** を使います。

```html
<button>ボタン1</button>
<button>ボタン2</button>

<div style="margin-top: 30px;">
  <button style="margin-right: 20px;">離れたボタン1</button>
  <button>離れたボタン2</button>
</div>

```

**イメージ:** 人と人との「ソーシャルディスタンス」です。隣とぶつからないように距離を取ります 🚶 ↔️ 🚶

## 🎨 最後のミッション：プロフィールサイト完成！

さあ、これまで学んだ「タグ」「画像」「色・配置」「余白」を全部使って、
世界に一つだけのプロフィールページを完成させましょう！

以下のコードをベースに、あなたの好きなように書き換えてください。

```html
<div style="text-align: center; padding: 20px;">

  <h1 style="color: dimgray; margin-bottom: 30px;">
    My Profile
  </h1>

  <img 
    src="https://images.unsplash.com/photo-1543466835-00a7907e9de1?auto=format&fit=crop&w=300&q=80" 
    style="border-radius: 50%; margin-bottom: 20px;" 
  />

  <p style="padding: 10px; background-color: #f0f0f0;">
    こんにちは！Webデザインを勉強して4日目です。<br>
    自分でコードを書いて動くのが楽しくなってきました。
  </p>

  <h2 style="color: white; background-color: orange; padding: 10px;">
    好きなこと
  </h2>
  
  <ul style="text-align: left;">
    <li style="margin-bottom: 10px;">カフェでまったりすること</li>
    <li style="margin-bottom: 10px;">美味しいパンを探すこと</li>
    <li>プログラミング（初心者！）</li>
  </ul>

</div>

```

<Callout type="warning">
**ちょっとした裏技 🪄**
**border-radius: 50%;** というコードを画像に入れると、画像が**丸く**切り取られます！
これを入れるだけで一気にアイコンっぽくなりますよ。
</Callout>
