## はじめに

前回は、文字に色をつけたり、真ん中に寄せたりして、ページがカラフルになりましたね！🎨

でも、まだ何かが足りない気がしませんか？
Webサイトでよく見る「プロフィールカード」や「ボタン」のような、**まとまり感**が少し弱いかもしれません。

第4回では、「枠線（ボーダー）」 と「角丸」という2つの魔法を使って、デザインをグッと引き締める方法を学びます。
これを使うだけで、ただの文字が「意味のあるパーツ」に見えてきますよ！

---

## 🖼️ 枠線（ボーダー）を引いてみよう

まずは、要素の周りに線を引いてみましょう。
**border**（ボーダー）というプロパティを使います。

以下のコードをエディタに貼り付けてみてください。

```html
<h1 style="border: 3px solid red;">
  赤枠のタイトル
</h1>

<p style="border: 1px solid black;">
  細い黒枠の文章です。
</p>

```

<Callout type="info">
文字の周りに四角い枠がつきましたか？
これがWebデザインの基本となる「ボックス（箱）」の形です 📦
</Callout>

### 3つの呪文を唱えよう

**border** を使うときは、3つのことをコンピュータに伝える必要があります。

**border: 太さ 種類 色;**

1. **太さ**: **1px**（細い）、**5px**（太い）など
2. **種類**: **solid**（実線）が基本ですが、**dotted**（点線）や **double**（二重線）もあります。
3. **色**: **red**, **black**, **gold** など

---

## 🪄 角を丸くして優しくしよう

四角い枠もいいですが、少しカクカクして「硬い」印象がありますよね。
今のWebサイトでは、角が丸いデザインが大人気です。

角を丸くするには **border-radius**（ボーダー・ラディアス）を使います。

```html
<h2 style="background-color: skyblue; border-radius: 10px;">
  角が丸いタイトル
</h2>

```

<Callout type="warning">
**変化が見えないときは？**
**border-radius** は、**背景色**（background-color）や**枠線**（border）がないと形が見えません。必ず色と一緒に使いましょう！
</Callout>

### 写真を丸く切り抜く裏技

この **border-radius**は、画像にも使えます。
なんと、**50%** という数字を指定すると、画像がまん丸になります！

```html
<img 
  src="https://placehold.co/200x200" 
  style="border-radius: 50%;" 
/>

```

---

## 📦 「透明な箱」でまとめよう

ここで新しいタグを紹介します。
** \<div\> **（ディブ）というタグです。

これ自体には何の意味もありません。ただの**「入れ物」**です。
でも、これこそがWebデザインの最重要アイテムなんです！

複数のタグを ** \<div\> ** で囲むと、ひとまとめにしてデザインできます。

```html
<div style="border: 2px solid orange; border-radius: 20px;">
  <h2>まとまった見出し</h2>
  <p>文章も一緒に囲まれています。</p>
  <p>これがグループ化です！</p>
</div>

```

サンドイッチをラップで包むように、中身をひとまとめにするイメージです 🥪

---

## 🧑‍💻 今日のミッション：プロフィールカード作成

さあ、今日学んだ「枠線」「角丸」「箱」を使って、おしゃれなプロフィールカードを作りましょう！

**目標:**

1. 全体を **\<div\>** で囲んで、枠線をつける
2. 画像を丸く切り抜く
3. 好きな色で飾る

```html
<div style="border: 4px solid skyblue; 
            border-radius: 20px; 
            text-align: center;">

  <h1 style="background-color: skyblue; color: white;">
    My Profile
  </h1>

  <img 
    src="https://images.unsplash.com/photo-1543466835-00a7907e9de1?auto=format&fit=crop&w=300&q=80" 
    style="border-radius: 50%;" 
  />

  <p>はじめまして！プログラミング勉強中です。</p>

  <h2>好きなこと</h2>
  <ul style="text-align: left;">
    <li>デザインすること</li>
    <li>新しい発見</li>
  </ul>

</div>

```

プレビューを見てみてください。
文字だけだった時よりも、ぐっと「Webサイトのパーツ」らしくなりませんか？✨
