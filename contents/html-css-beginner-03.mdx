## はじめに

前回までで、文字や写真を表示できるようになりました。
でも……まだ背景は真っ白で、文字は黒いまま。ちょっと「事務的な書類」っぽいですよね。

そこで登場するのが **CSS（シーエスエス）** です！
HTMLが「骨組み」なら、CSSは「お洋服」や「メイク」。

今回は、一番カンタンな方法で、ページをカラフルに可愛くしてみましょう！💄

---

## 🎨 色を変える魔法「style」

タグの中に **style=\"...\"** というおまじないを書くと、そこだけデザインを変えることができます。

まずは、文字の色を変えてみましょう。
以下のコードをコピーして、エディタに貼ってみてください。

```html
<h1 style="color: pink;">私のプロフィール</h1>
<p style="color: skyblue;">こんにちは！</p>
```

<Callout type="info">
文字がピンクや水色になりましたか？✨
`color: 色の名前;` と書くだけで、好きな色に変えられるんです！
</Callout>

### 使える色の名前
**red** (赤)、**blue** (青)、**green** (緑) などの英語はもちろん、こんな色も使えます：
- **orange** (オレンジ)
- **purple** (紫)
- **gold** (金色)
- **gray** (灰色)

---

## 🖌️ 背景に色を塗ってみよう

文字の色だけでなく、背景の色も変えられます。
**background-color** （バックグラウンド・カラー）を使います。

```html
<h1 style="background-color: yellow;">目立つタイトル</h1>
```

文字の色と組み合わせることもできますよ！
（命令を `;` で区切ってつなげます）

```html
<h1 style="background-color: black; color: white;">
  黒背景に白文字でクールに！
</h1>
```

---

## 📏 真ん中に寄せてみよう

タイトルや画像を、ページの真ん中に置きたいときってありますよね。
そんな時は **text-align: center;** （テキスト・アライン：センター）を使います。

```html
<h1 style="text-align: center;">真ん中のタイトル</h1>
```

画像も同じように真ん中にできますが、ちょっとコツがいるので、まずは `<p>` タグで囲んでみましょう。

```html
<p style="text-align: center;">
  <img src="https://placehold.co/200x200" />
</p>
```

---

## 👗 今日のミッション：おしゃれなプロフィール作成

さあ、これまでの知識を総動員して、「誰かに見せたくなるプロフィール」を作りましょう！

**目標:**
1. タイトルを真ん中に寄せて、好きな色にする。
2. 背景色をつけて、マーカーを引いたように見せる。
3. リストも忘れずに！

**お手本コード:**

```html
<h1 style="text-align: center; color: hotpink;">
  My Profile
</h1>

<p style="text-align: center;">
  <img src="[https://images.unsplash.com/photo-1490750967868-58cb75069ed6?auto=format&fit=crop&w=300&q=80](https://images.unsplash.com/photo-1490750967868-58cb75069ed6?auto=format&fit=crop&w=300&q=80)" />
</p>

<h2 style="background-color: lemonchiffon;">
  🌸 自己紹介
</h2>
<p>はじめまして！Webデザイン勉強中です。</p>

<h2 style="background-color: lavender;">
  🍰 好きなもの
</h2>
<ul>
  <li>チョコレート</li>
  <li>猫の動画を見ること</li>
</ul>
```

<Callout type="warning">
**書き方のコツ**
**style=" ... "** のダブルクォーテーション（ちょんちょん）を忘れないように気をつけて！
</Callout>
