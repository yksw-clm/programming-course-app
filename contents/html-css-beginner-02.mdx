## はじめに

前回は「文字」を表示させましたね。でも、文字だけだと少し寂しくないですか？

第2回では、ページに**「写真」**を貼ったり、**「箇条書き」**で情報を整理したりして、一気にWebサイトっぽく変身させます。

「百聞は一見にしかず」。写真を1枚貼るだけで、ページの雰囲気がガラッと変わりますよ！✨

---

## 📸 写真を貼ってみよう

Webサイトに画像を貼るには、`<img>`（イメージ）というタグを使います。

まずは、以下のコードをコピーして、エディタに貼り付けてみてください。

```html
<h2>私のお気に入り</h2>
<img src="https://images.unsplash.com/photo-1543466835-00a7907e9de1?auto=format&fit=crop&w=300&q=80" />
```

<Callout type="info">
かわいい犬の写真が表示されましたか？🐶
</Callout>

### どういう仕組み？

```html
<img src="画像の住所(URL)" />
```

- **`img`**: Image（画像）の略です。
- **`src`**: Source（ソース＝情報の出どころ）の略です。「ここに画像があるよ！」と教えてあげています。

<Callout type="warning">
**不思議なポイント 😲**
前回の `<h1>...</h1>` と違って、`<img>` には **終わりのタグ（閉じタグ）がありません**。
「中身を挟む」必要がないので、これだけでOKなんです。
</Callout>

---

## 📝 箇条書き（リスト）を作ろう

次は、趣味や好きなものを「リスト」にして見やすく整理しましょう。
ただ並べるよりも、頭に「・」がついていると読みやすいですよね。

以下のコードをコピーして貼り付けてみてください。

```html
<h2>私の趣味リスト</h2>
<ul>
  <li>カフェ巡り</li>
  <li>映画鑑賞</li>
  <li>旅行</li>
</ul>
```

### どういう仕組み？

ここでは2種類のタグがチームワークで動いています。

1. **`<ul>`**: Unordered List（順序のないリスト）の略。
   「ここからリストが始まるよ～」という**枠組み**です。
2. **`<li>`**: List Item（リストの項目）の略。
   実際の**中身**です。

```html
<ul>  ← ここからリスト！
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ul> ← ここまで！
```

サンドイッチで言うと、`<ul>` がパンで、`<li>` が具材です。具材はいくつ入れてもOKです！🥪

---

## 🧑‍💻 今日のミッション：プロフィール完成！

それでは、今日学んだことを組み合わせて、あなたのプロフィールをリッチにしましょう。

以下の手順でコードを書いてみてください。

1. **`<h1>`** で名前を書く
2. **`<img>`** で好きな画像を貼る
   （練習用に `https://placehold.co/300x200` というURLを使うと、グレーの四角が表示されます！）
3. **`<ul>`** と **`<li>`** で「好きな食べ物ベスト3」を書く

**完成例のイメージ:**

```html
<h1>山田 花子のプロフィール</h1>

<img src="[https://images.unsplash.com/photo-1543466835-00a7907e9de1?auto=format&fit=crop&w=300&q=80](https://images.unsplash.com/photo-1543466835-00a7907e9de1?auto=format&fit=crop&w=300&q=80)" />

<h2>好きな食べ物</h2>
<ul>
  <li>焼肉</li>
  <li>お寿司</li>
  <li>パンケーキ</li>
</ul>
```

プレビューを見て、それっぽくなっていたら大成功です！🎉
